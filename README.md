# 校园宿舍管理系统 java249

## 项目概述

校园宿舍管理系统是基于 Spring Boot 和 Vue.js 技术栈开发的一个应用系统。该系统实现了宿舍管理的基本功能，支持前后端分离的架构，代码完整，已经过调试，确保可以正常运行。

## 技术栈

- 后端：Spring Boot
- 前端：Vue.js

## 功能模块

- 学生信息管理
  - 学生基本信息录入
  - 学生住宿信息维护
- 宿舍楼管理
  - 宿舍楼信息维护
  - 宿舍房间分配管理
- 消息通知
  - 住宿通知发布
  - 通知历史记录查询
- 维修服务
  - 维修请求提交
  - 维修进度跟踪

## 系统角色

- 系统管理员
- 宿舍管理员
- 学生用户

## 运行环境

- 后端
  - Java 1.8 或以上版本
  - MySQL 5.7 或以上版本
- 前端
  - 现代浏览器（建议使用 Chrome、Firefox）

## 部署步骤

1. 准备后端运行环境：安装 Java 和 MySQL
2. 初始化数据库：运行数据库脚本，建立必要的表结构
3. 配置后端应用：修改配置文件中的数据库连接信息
4. 部署后端服务：通过 Spring Boot 打包应用，运行 jar 包
5. 部署前端应用：在 Web 服务器上部署编译后的前端资源
